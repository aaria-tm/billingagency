# Billing Software
Helps small agencies to generate bills easily, one click and you're done. Easy to put the items in, add quantity, autosums the total cost. 

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

## 🚀 Features
-Create Bill / Invoice:
   1. Add items, quantity, price
   2. Auto total calculation

-Auto Calculations:
   1. Subtotal
   2. Tax  (GST if you want)
   3. Final amount
      
-Download / Print Bill:
   1. Generate printable invoice

## 📂 How to Run Locally
1. Download or clone the repo
2. Open index.html in your browser
